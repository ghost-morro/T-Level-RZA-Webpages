# Ridget Zoo Adventures Website

## Overview

Ridget Zoo Adventures is a full stack web development project created as part of my Digital Production, Design and Development T Level coursework.

The aim of the project was to design and develop a functional website that allows users to access information about Ridget Zoo Adventures, purchase tickets, make hotel reservations, and manage user accounts through login and registration systems.

## Features

* Homepage with navigation and website information
* User login and account creation system
* Ticket ordering functionality
* Hotel reservation system
* Front end website design and layout
* Back end functionality developed using PHP

## Technologies Used

* HTML
* CSS
* PHP

## Skills Demonstrated

* Front end development
* Back end development
* User interface design
* Website structure and navigation
* Problem solving
* Functional web development

## Project Purpose

This project was developed to strengthen my understanding of both front end and back end web development, while applying practical programming and design skills in a realistic scenario.
